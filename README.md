# mp-tasker
Add a Task for the World.
Just some play with Node.js Angular.js and Mongo

### Dependencies
* node.js
* angular.js
* express       
* mongoose       
* morgan         
* body-parser    
* method-override

### Installation
- `npm install`
- copy `config/config.js.dist` to `config/config.js`
- edit `config/config.js` to add your own mongodb url
- run `node server.js`
